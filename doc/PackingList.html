Decaf Prototype Packing List

CD Contents

  • decaf_binaries.zip - Prototype binaries (contents below)
  • devcpp_install.exe - Installer for Dev-Cpp
  • JRE1-4-1_install.exe - Installer for Java version 1.4.1
  • LICENSE - The license terms of this prototype
  • readme.html - includes installation instructions for the prototype binaries
  • decaf/ - The sources and libraries required to run Decaf
    This is a reference copy only; it cannot be run from the CD. Please install the prototype before running it.
  • doc/ - Documentation for Decaf and uMail
  • api/ - API documentation for Decaf, and source documentation for uMail
  • jdk/ - API documentation for Java 1.4.1
  • example/ - Java source code samples of Decaf editor customization
  • umail/ - C++ source code and Dev-Cpp project files for the demo (native) application uMail
